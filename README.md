# Multi-Agent AI Research System

A sophisticated multi-agent AI system designed for collaborative research, analysis, and autonomous problem-solving using AI agents. The system enables specialized agents to work together seamlessly for information gathering, reasoning, critique, and report generation.

---

## Features

- Multi-Agent AI Architecture
- Automated Web Research Pipeline
- Intelligent Task Distribution
- AI-Based Report Generation
- Critic & Validation Workflow
- Extensible Agent Design
- Environment Variable Support
- Modular Python Codebase

---

## Tech Stack

- Python
- LangChain 
- OpenAI / Ollama Models
- Rich (CLI Visualization)
- Environment Variables with `python-dotenv`
